Isso é um uma parte do texto em negrito, usando semântica,que serve para destacar uma palavra forte no texto!

O Itálico serve para dar ênfase à algo, tudo isso usando semântica!

támbem temos a tag MARK, que serve como marcador de textos!

Para fazer letras miúdas/pequenas, você pode usar a tag SMALL, muito usada támbem em contratos!

Você támbem pode usar a tag DEL para sinalizar que aquele texto deve ser lido, mas não considerado

Podemos támbem usar a tag INS, para dar destaque no texto, e sinalizar que aquele texto foi inserido depois!

Podemos usar a tag SUP para indicar textos sobrescritos como Exemplo: X3-23=122, e usar a tag SUB para textos subscritos, como Exemplo: H2O
